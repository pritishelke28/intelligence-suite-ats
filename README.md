# 🚀 Intelligence Suite: AI-Powered ATS

An advanced Applicant Tracking System (ATS) built with **Flask** and **Natural Language Processing**. This platform automates the recruitment workflow by scoring resumes against job descriptions using TF-IDF vectorization and Cosine Similarity.

## ✨ Features

### 👤 For Candidates
- **Smart Application:** Upload resumes and provide professional metadata (CTC, Notice Period).
- **AI Analysis:** View a "Fit Score" and detailed skill gap analysis for every application.
- **Dynamic Dashboard:** Track application status (Pending, Shortlisted, Rejected) in real-time.

### 🛠️ For Administrators
- **Mission Control:** Centralized dashboard to monitor all jobs and applicant volume.
- **Deep-Dive Review:** Access automated intelligence reports for every candidate.
- **Job Lifecycle:** Create, edit, and toggle job postings between 'Open' and 'Closed'.

## 🛠️ Tech Stack
- **Backend:** Python / Flask
- **AI/ML:** Scikit-learn (TF-IDF), NLTK (Text Preprocessing)
- **Database:** SQLite
- **Parsing:** PyPDF2 / pypdf
